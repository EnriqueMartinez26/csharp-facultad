<p align="center">
  Trabajo practico para la materia Laboratorio, desarrollado durante la tecnicatura y organizado como un conjunto de practicas de consola y Windows Forms.
</p>

## Estructura

| Carpeta | Enfoque |
| --- | --- |
| `practica-delegados` | Delegados y funciones como parametro |
| `practica-eventos` | Eventos y comunicacion entre objetos |
| `practica-datagridview` | Uso basico de `DataGridView` |
| `crud-datagridview` | CRUD con persistencia local |
| `practica-poo` | Programacion orientada a objetos |
| `JuegoPreguntas` | Juego de preguntas con datos en JSON |

## Stack

- C#
- .NET
- Windows Forms
- Consola
- JSON

## Lectura rapida

- `practica-delegados` y `practica-eventos` agrupan ejercicios de sintaxis y comunicacion entre objetos.
- `practica-datagridview` y `crud-datagridview` trabajan con interfaces WinForms y manejo de datos.
- `practica-poo` reúne dos soluciones de consola con modelado por capas.
- `JuegoPreguntas` centraliza una practica de consola con carga de datos desde JSON.
