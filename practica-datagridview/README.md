# Práctica de DataGridView

Trabajo práctico para la materia Laboratorio, desarrollado en Windows Forms durante la tecnicatura para ejercitar el uso de `DataGridView` y operaciones básicas sobre conjuntos de datos.

El proyecto se utilizó como práctica de carga, consulta, filtrado y cálculo de información en una interfaz de usuario de escritorio.

## Funcionalidades

- Agregar personas al `DataGridView`
- Eliminar filas seleccionadas
- Buscar coincidencias por nombre
- Filtrar mayores de edad
- Calcular promedio de edades

## Tecnologías

- C#
- Windows Forms
- .NET Framework 4.8
