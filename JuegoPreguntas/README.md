# Juego de Preguntas

Trabajo práctico para la materia Laboratorio, desarrollado en C# durante la tecnicatura y orientado a la lectura de datos desde JSON, el manejo de puntuación y el control del flujo del juego.

El proyecto permite ejercitar estructuración básica de una aplicación, separación en capas simples y manipulación de colecciones en memoria.

## Estructura

- `Models`
- `Services`
- `Data/preguntas.json`

## Funcionalidades

- Carga de preguntas desde JSON
- Selección aleatoria de categorías
- Puntuación por rondas
- Registro de respuestas del jugador

## Tecnologías

- C#
- .NET 8
- Consola
- JSON
