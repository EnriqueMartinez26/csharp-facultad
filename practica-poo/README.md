# Práctica de POO

Trabajo práctico para la materia Laboratorio, con dos soluciones de consola desarrolladas en C# durante la tecnicatura para ejercitar Programación Orientada a Objetos y separación básica por capas.

Las soluciones reúnen ejercicios y estructuras simples que permiten revisar encapsulamiento, clases, colecciones y colaboración entre componentes.

## Soluciones incluidas

### PracticaPoo

Solución de consola con una biblioteca compartida y un menú que organiza distintos casos de uso vinculados a conceptos básicos de POO.

- `PracticaPoo.Console`
- `PracticaPoo.Lib`
- `net9.0`

### GestionEmpleados

Solución de consola con una jerarquía de empleados y una biblioteca de modelos para practicar abstracción, herencia e implementación de comportamientos compartidos.

- `GestionEmpleados.Console`
- `GestionEmpleados.Lib`
- `net9.0`

## Tecnologías

- C#
- .NET 9
- Consola
- Librerías compartidas
- POO
