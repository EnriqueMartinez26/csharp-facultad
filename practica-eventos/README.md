# Práctica de Eventos

Trabajo práctico para la materia Laboratorio, desarrollado en C# durante la tecnicatura y enfocado en el uso de eventos y mecanismos de comunicación entre objetos.

Las actividades incluidas permiten practicar la publicación y suscripción de eventos, el uso de `EventArgs` y la organización de lógica reactiva en aplicaciones simples.

## Contenido

- Evento personalizado simple
- Sensor de temperatura
- Varios suscriptores
- Evento con `EventArgs`
- Simulación de descarga
- Evento y lógica de negocio
- Control de acceso
- Encadenamiento de eventos
- Menú interactivo por consola
- Contador de eventos

## Tecnologías

- C#
- .NET 9
- Consola
