# Práctica de Delegados

Trabajo práctico para la materia Laboratorio, desarrollado en C# durante la tecnicatura y orientado al estudio de delegados y patrones básicos de invocación de métodos.

El objetivo de estas prácticas es consolidar conceptos vinculados a firmas de métodos, paso de funciones como parámetros, filtros y uso de funciones anónimas en escenarios sencillos.

## Contenido

- Delegado básico
- Varios métodos con el mismo delegado
- Delegado como parámetro de un método
- Delegado con `void`
- Multicast de métodos
- Expresión lambda con delegado
- Formateador de texto con delegado
- Filtro con delegado
- Delegado como propiedad

## Tecnologías

- C#
- .NET 9
- Consola
