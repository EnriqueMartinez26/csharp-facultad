# CRUD de DataGridView

Trabajo práctico para la materia Laboratorio, desarrollado en Windows Forms durante la tecnicatura para practicar operaciones CRUD con persistencia local.

El trabajo aborda alta, baja, modificación y consulta de registros, junto con una persistencia simple en JSON y búsqueda sobre los datos cargados.

## Funcionalidades

- Alta de registros con validación básica
- Eliminación de registros
- Búsqueda en tiempo real
- Persistencia automática en `datos.json`
- Uso de `DataGridView` para visualizar datos

## Tecnologías

- C#
- Windows Forms
- .NET Framework 4.8
- Newtonsoft.Json
